## 文档规范

所有项目报告、技术文档都应放在此 `docs/` 目录中。

### 命名规范

- 使用英文文件名
- 使用大写下划线格式（`EXAMPLE_REPORT.md`）
- 或驼峰格式（`ExampleReport.md`）

### 文档类型

- 开发进度报告
- 技术设计文档
- API 文档
- 测试报告
- 用户手册
